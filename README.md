# FileManager Plus

A premium, feature-rich file manager for Android with a focus on privacy, aesthetics, and organization.

## Key Features

- **Smart Locker**: Secure your sensitive files with a 4-digit PIN. Hidden directory (`.Locker`) with `.nomedia` support.
- **Smart Restore**: Locked files remember their original folder and are restored exactly where they were.
- **Secure File Viewing**: View encrypted Images and PDFs directly within the app's internal viewers.
- **Advanced File Operations**: Copy, Cut, Delete, Rename, and Share with multi-selection support.
- **Archive Support**: Open, extract, and create ZIP archives smoothly.
- **App Management**: View installed apps, clear cache (automated), and uninstall directly.
- **Modern UI/UX**:
    - **Custom Themes**: Dark, Light, and OLED (Pure Black) modes.
    - **Premium Animations**: Smooth list transitions and screen sliding effects.
    - **Persistent Search**: Quick access to your files across the system.
- **Permissions Dashboard**: Easily manage critical system permissions (MANAGE_EXTERNAL_STORAGE, etc.).

## Installation

1. Install the APK on your Android device (Android 7.0+ / API 24).
2. Grant the **All Files Access** permission when prompted for full functionality.
3. For automated cache cleaning, enable the **Accessibility Service** in settings.

## Privacy & Security

- All locked files are stored in a hidden system directory.
- The Locker PIN is stored locally and securely using Android SharedPreferences.
- No external servers are used; your data stays on your device.
